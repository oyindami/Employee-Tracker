# Employee-Tracker

This Application is used to view, add and delete employees


## User Story

```md
AS A business owner
I WANT to be able to view and manage the departments, roles, and employees in my company
SO THAT I can organize and plan my business
```

## Technolgies used 
Inquirer
console.table
MySQL2
